# T3rn-Executor
## How to running
1. ``git clone https://github.com/dwisyafriadi2/t3rn-executor.git``
2. ``cd t3rn-executor``
3. ``chmod +x autoinstall.sh``
4. ``chmod +x cek-status.sh``
5. ``chmod +x stop-executor.sh``
6. ``chmod +x start-executor.sh``
7. ``./autoinstall.sh``
